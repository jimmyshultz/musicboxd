Use archive documents when:
- Understanding why a decision was made
- Looking for historical context
- Learning from past implementation approaches
- Debugging issues that may relate to past changes

Do not use archive documents for:
- Current implementation guidance
- Production deployment procedures
- Active feature development
- Current architecture decisions
