Reporting Bugs

Found a bug? We want to know about it! Reporting bugs helps us fix issues quickly and improve the app for everyone.

When reporting a bug, please include:

  • What happened: A clear description of the issue
  • When it happened: What you were doing when the bug occurred
  • Steps to reproduce: Detailed steps so we can reproduce the issue
  • Expected behavior: What you expected to happen
  • Actual behavior: What actually happened
  • Device information: Your device model and iOS version (if applicable)
  • Screenshots or screen recordings: Visual evidence is very helpful!

Report bugs to: jimmyshultz3@gmail.com

Please include "Bug Report" in your subject line.
